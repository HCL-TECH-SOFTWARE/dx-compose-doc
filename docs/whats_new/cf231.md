# What's new in CF231

The following features and updates are available to customers installing HCL Digital Experience (DX) Compose Container Update CF231 on supported platforms:

- Search V2 - New Atomic Components and supported CSS part attributes available
- Site Builder is now available in HCL DX Compose

## Search V2 - New Atomic Components and supported CSS part attributes available

New Atomic Components (ACs) and their corresponding CSS part attributes are now available for Search V2. Existing ACs have also been updated to support additional CSS part attributes. For more information, refer to [Search V2 - Atomic Components](https://pages.git.cwp.pnp-hcl.com/CWPdoc/dx-mkdocs/in-progress/build_sites/search_v2/components/#atomic-components){target="_blank"}.

## Site Builder is now available in HCL DX Compose

You can now use the Site Builder application to create site or section templates for use in HCL DX Compose. For more information on Site Builder and how to import its template library onto a new virtual portal, refer to the following topics:

- [Site Builder](https://help.hcl-software.com/digital-experience/9.5/latest/build_sites/sitebuilder/){target="_blank"}
- [Importing the Site Builder Template library onto a new virtual portal](../deploy_dx/manage/cfg_webengine/import_site_builder_library.md)
